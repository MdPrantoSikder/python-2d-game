# 🎮 Python 2D Platformer

A classic 2D platformer built using **Python** and **Pygame**, inspired by *Metroidvania* design principles. This project focuses on core game development concepts such as physics, collision detection, and modular architecture—making it ideal for learning and experimentation.

---

## 🚀 Features

* 🎯 **Responsive Player Controls**
  Smooth movement, jumping, and gravity-based physics

* 🧱 **Tile-Based Collision System**
  Efficient and scalable level design using grid-based maps

* 🤖 **Basic Enemy AI**
  Simple enemy behaviors to create gameplay challenges

* 💎 **Collectibles System**
  Items that enhance exploration and engagement

* 🌍 **Multi-Level Progression**
  Structured levels with increasing difficulty

* 🧩 **Modular Code Design**
  Easily extendable for adding new mechanics and features

---

## 🧠 Learning Objectives

This project helps you understand:

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/python-2d-game.git
cd python-2d-game
```

2. Install dependencies:

```bash
pip install pygame
```

3. Run the game:

```bash
python src/main.py
```
